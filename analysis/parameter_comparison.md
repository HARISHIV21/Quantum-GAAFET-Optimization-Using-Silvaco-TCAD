# Parameter Comparison

## Baseline vs Optimized GAAFET

| Parameter | Baseline | Optimized | Purpose |
|-----------|----------|-----------|---------|
| Channel Length | 200 nm | 300 nm | Reduce short-channel effects |
| Radius | 25 nm | 25 nm | Maintain nanowire geometry |
| Oxide Thickness | 45 nm | 6 nm | Improve gate electrostatic control |
| Source Doping | 1×10¹⁹ cm⁻³ | 5×10¹⁸ cm⁻³ | Reduce leakage current |
| Drain Doping | 1×10¹⁹ cm⁻³ | 5×10¹⁹ cm⁻³ | Improve ON-state current |
| Channel Doping | 1×10¹⁷ cm⁻³ | 5×10¹⁶ cm⁻³ | Enhance carrier transport |

## Summary

The optimized GAAFET improves electrostatic control through oxide scaling, channel engineering, and optimized doping, resulting in lower leakage current and improved switching performance.
